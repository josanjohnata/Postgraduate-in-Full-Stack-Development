O que é HTML?

Hiper Text Markup Language

- Linguagem padrão criada para páginas da Web.
- Criada em 1991 pelo britânico Tim Berners-Lee e mantida pela W3C (World Wide Web Consortium)
- Descreve a estrutura básica do site.
- Consiste em uma série de elementos que informam ao navegador como exibir o conteúdo
- Os elementos definem partes do conteúdo como "este é um título", "este é um parágrafo", "este é um link", etc'

Estrutura básica de um código HTML

Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final.

Tag iniciais:
- Título
  <h1>Meu primeiro título</h1>

- Parágrafo
  <p>Meu primeiro parágrafo.</p>

- Link
  <a href="https://www.google.com.br">Google</a>

Elementos semânticos = elementos com significado

Um elemento semântico descreve claramente seu significado tanto para o navegador quanto para o desenvolvedor.
* Antes do HTML5:
  - <div id="header">
  - <div id="nav">
  - <div id="maincontent">

O elemento <article> especifica um conteúdo independente.
O elemento <section> define a seção em um documento.
